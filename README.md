# Data Cleaning & Visualization Project

## Overview
This project focuses on cleaning and processing raw data and creating visual representations to gain meaningful insights. The main objective is to understand how real-world datasets can contain missing values, duplicates, and inconsistencies, and how these issues can be handled using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Load and process raw dataset
- Handle missing values
- Remove duplicate records
- Detect outliers
- Generate visualizations
- Analyze correlations between variables

## Dataset Information
The dataset used in this project contains student-related information such as:

- Student ID
- Age
- Study Hours
- Attendance
- Marks

The dataset also includes missing values for practicing data preprocessing techniques.

## Visualizations Included
- Histograms
- Boxplots
- Correlation Heatmap

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/naman123aS/Data-Cleaning-Visualization-Project.git
```

2. Move to project directory:

```bash
cd Data-Cleaning-Visualization-Project
```

3. Install required libraries:

```bash
pip install pandas matplotlib seaborn
```

4. Run the project:

```bash
python data_cleaning_project.py
```
